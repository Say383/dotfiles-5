1 Installing with Ansible

1.1 Requirements

  • Ansible

Install Ansible galaxy requirements with ansible-galaxy install -r requirements.yml

1.2 OSX

For MacOS, run ansible-playbook -i inventory osx.yml

1.3 Linux

For Linux, run ansible-playbook -i inventory archlinux.yml

2 Installing with Chezmoi

2.1 Requirements

  • Chezmoi

2.2 Installing

Run chezmoi init https://github.com/benmezger/dotfiles.git (see reference for more options) to checkout the repository and any submodules. Finally, run chezmoi apply to move changes to your home directory.

When installing without Ansible, chezmoi wont install any external dependencies, so make sure you have all the required dependencies before runnign chezmoi apply.
